# 참고 논문 및 자료

## 핵심 추천 논문 (2024~2025)

### TradingAgents: Multi-Agents LLM Financial Trading Framework
- **발표**: 2024년 12월, arXiv
- **내용**: 기본 분석가, 심리 분석가, 기술 분석가, 황소/곰 토론자, 리스크 관리자 등 다양한 역할의 LLM 에이전트가 회사를 이루어 주식 투자하는 프레임워크
- **링크**: [arXiv:2412.20138](https://arxiv.org/abs/2412.20138)
- **GitHub**: [TauricResearch/TradingAgents](https://github.com/TauricResearch/TradingAgents)

### Behavioral Consistency Validation for LLM Agents
- **부제**: An Analysis of Trading-Style Switching through Stock-Market Simulation
- **발표**: 2025~2026년
- **내용**: 행동 재무학 이론(손실 회피, 군집 행동 등)을 LLM 성격 파라미터로 주입 후 1년간 모의 주식 시장 시뮬레이션 실험
- **링크**: [arXiv:2602.07023](https://arxiv.org/abs/2602.07023)

### Persona-Driven LLM Interaction in Stock Market Simulations
- **발표**: 2025년, OpenReview
- **내용**: LLM에 '위험 선호', '경쟁적', '적응형' 등 서로 다른 페르소나 부여 후 주식 시장 시뮬레이션 상호작용 연구
