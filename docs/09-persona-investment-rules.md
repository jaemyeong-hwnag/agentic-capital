# 페르소나 기반 투자 규칙

페르소나의 성격 파라미터가 투자 행동의 모든 측면을 자동으로 결정한다.

---

## 1. 자본 배분 (Capital Allocation)

각 페르소나는 자신에게 배정된 **현재 자산의 일부만** 투자할 수 있다.

### 배분 구조

```
회사 총 자본
 └─ CEO가 각 에이전트에게 배정 (에이전트별 운용 자산)
     └─ 에이전트는 운용 자산의 일부만 1회 투자 가능
```

### 1회 투자 한도 (성향별)

| 위험선호도 | 1회 최대 투자 비율 | 예시 (운용자산 $10,000) |
|-----------|-------------------|----------------------|
| 매우 보수적 (0~20) | 운용자산의 5% | $500 |
| 보수적 (21~40) | 운용자산의 10% | $1,000 |
| 중립 (41~60) | 운용자산의 15% | $1,500 |
| 공격적 (61~80) | 운용자산의 25% | $2,500 |
| 매우 공격적 (81~100) | 운용자산의 40% | $4,000 |

### 시스템 강제 제한 (Guardrail)

- **절대 한도**: 어떤 성향이든 운용자산의 50% 초과 불가
- **최소 예비금**: 운용자산의 20%는 항상 현금 보유 필수
- **동시 포지션 제한**: 최대 보유 종목 수 = `ceil(위험선호도 / 20)` (1~5개)

## 2. 투자 시간 (Trading Timing)

페르소나의 성격이 투자 타이밍을 결정한다.

### 성실성(Conscientiousness) 기반

| 성실성 수준 | 투자 패턴 | 설명 |
|-----------|----------|------|
| 높음 (70~100) | 정해진 스케줄 엄수 | 매일 정시에 시장 분석 후 판단 |
| 중간 (30~69) | 유연한 스케줄 | 대체로 규칙적이나 시장 변동 시 추가 판단 |
| 낮음 (0~29) | 불규칙 | 기분에 따라 수시로 판단, 때로 며칠 방치 |

### 신경증(Neuroticism) 기반 반응성

| 신경증 수준 | 시장 급변 시 반응 |
|-----------|----------------|
| 높음 | 즉시 반응 (패닉셀/충동매수 가능성) |
| 중간 | 1~2시간 후 냉정하게 분석 후 판단 |
| 낮음 | 일별 정기 리뷰에서만 반응 |

### 외향성(Extraversion) 기반 정보 수집

| 외향성 수준 | 판단 소스 |
|-----------|----------|
| 높음 | 뉴스, SNS 감성, 다른 에이전트 의견 적극 수렴 |
| 낮음 | 차트/재무제표 등 정량 데이터만 활용 |

---

## 3. 종목 선택 (Asset Selection)

### 개방성(Openness) 기반

| 개방성 수준 | 종목 성향 | 예시 |
|-----------|----------|------|
| 높음 (70~100) | 신규/트렌드 종목 선호 | 밈코인, IPO, 신기술 섹터 |
| 중간 (30~69) | 균형 | 성장주 + 가치주 혼합 |
| 낮음 (0~29) | 검증된 종목만 | 대형주, 배당주, 인덱스 ETF |

### 손실회피(Loss Aversion) 기반

| 손실회피 수준 | 포트폴리오 구성 |
|-------------|---------------|
| 높음 | 채권/금 비중 높음, 손절 라인 타이트 (-3~5%) |
| 중간 | 주식 60% + 안전자산 40%, 손절 -10% |
| 낮음 | 공격적 주식/레버리지 가능, 손절 느슨 (-20% 이상) |

### 위험선호도(Risk Appetite) 기반 시장 선택

| 위험선호도 | 선호 시장 |
|-----------|----------|
| 보수적 | 국내 대형주, 미국 S&P500 ETF |
| 중립 | 미국 개별주, 국내 중소형주 |
| 공격적 | 암호화폐, 선물/옵션, 소형 알트코인 |

## 4. 매매 전략 (Trading Strategy)

### 성격 조합별 전략 프로필

| 프로필 | 성격 조합 | 전략 |
|--------|----------|------|
| **가치 투자자** | 성실성↑ 개방성↓ 신경증↓ | 장기 보유, 재무제표 분석, 저평가 종목 |
| **모멘텀 트레이더** | 외향성↑ 개방성↑ 신경증↑ | 뉴스/트렌드 추종, 빠른 진입/퇴장 |
| **스윙 트레이더** | 성실성↑ 위험선호 중립 | 기술적 분석, 수일~수주 보유 |
| **배당 수집가** | 우호성↑ 손실회피↑ 성실성↑ | 고배당 종목, 분기별 리밸런싱 |
| **투기꾼** | 개방성↑ 위험선호↑ 성실성↓ | 단타, 레버리지, 밈코인 |

### 보유 기간 결정

```
기본_보유일 = 성실성 * 0.5 + 손실회피 * 0.3   (1~50일 범위)
변동성_민감도 = 신경증 * 0.01                   (0~1 범위)

실제_보유일 = 기본_보유일 * (1 - 시장변동률 * 변동성_민감도)
```

## 5. 감정 상태 시스템 (Emotional State)

페르소나는 고정된 성격 외에 **변동하는 감정 상태**를 가진다.

### 감정 파라미터 (Redis에 실시간 저장)

| 파라미터 | 범위 | 영향 |
|---------|------|------|
| **스트레스** | 0~100 | 높으면 손절/관망 확률 증가 |
| **자신감** | 0~100 | 높으면 투자 비율/빈도 증가 |
| **피로도** | 0~100 | 높으면 판단 빈도 감소, 실수 확률 증가 |

### 감정 변화 트리거

| 이벤트 | 스트레스 | 자신감 | 피로도 |
|--------|---------|--------|--------|
| 수익 +5% 이상 | -10 | +15 | - |
| 손실 -5% 이상 | +20 | -15 | +5 |
| 연속 3회 수익 | -5 | +20 | - |
| 연속 3회 손실 | +30 | -25 | +10 |
| CEO 경고 받음 | +25 | -20 | +15 |
| 승진 | -15 | +25 | -10 |
| 해고 경고 | +40 | -30 | +20 |

### 감정 → 행동 매핑

- **스트레스 > 80**: 투자 판단 전 "냉각 기간" 강제 적용 (1턴 스킵)
- **자신감 > 90**: 과잉 투자 위험 → Guardrail에서 투자 한도 10% 하향
- **피로도 > 70**: 판단 횟수 하루 절반으로 제한

## 6. 에이전트 간 상호작용

### 우호성(Agreeableness) 기반

| 우호성 수준 | 행동 |
|-----------|------|
| 높음 | 다른 에이전트의 추천 종목 수용, 공동 투자 참여 |
| 낮음 | 독자적 판단, 반대 포지션 선호 (역발상 투자) |

### 직급별 영향력

| 직급 | 권한 |
|------|------|
| CEO | 전체 자본 배분, 에이전트 채용/해고, 투자 방향 설정 |
| 시니어 | 운용자산 多, 주니어에게 조언 가능, 종목 추천 |
| 주니어 | 운용자산 少, 시니어 조언 수용 여부는 성격에 따라 |
| 인턴 | 최소 운용자산, 모의투자만 가능, 관찰 및 학습 |

## 7. 포트폴리오 제약 조건 (전체 시스템)

| 제약 | 값 | 설명 |
|------|-----|------|
| 단일 종목 집중 한도 | 전체 자본의 30% | 한 종목에 과도한 집중 방지 |
| 일일 전체 거래 한도 | 에이전트당 최대 5회 | API 과부하 및 핑퐁 방지 |
| 최소 보유 시간 | 5분 (암호화폐) / 1일 (주식) | 초단타 방지 |
| 레버리지 제한 | 위험선호 80 이상만 허용, 최대 2배 | 파산 방지 |
| 동일 종목 재진입 대기 | 매도 후 최소 1턴 대기 | 감정적 재매수 방지 |
