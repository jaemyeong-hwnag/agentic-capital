# 프로젝트 개요: AI 페르소나 기반 투자 시뮬레이션

## 핵심 컨셉

AI 에이전트 각각에 인간의 성향/성격/심리 특성을 부여하고, 가상의 펀드 회사를 운영하는 시뮬레이션 프로젝트.

## 동작 방식

- **역할 놀이 구조**: 각 AI 에이전트가 DB에 저장된 성향/성격/심리 컨텍스트를 기반으로 역할을 수행
- **펀드 회사 운영**: CEO AI 1명 + 랜덤 성향의 직원 AI들로 구성
- **자본 배분**: CEO가 각 에이전트에게 운용자산 배정, 에이전트는 **운용자산의 일부만** 투자 가능
- **자동 결정**: 페르소나의 성격 파라미터가 투자 시간, 종목, 전략, 비율을 자동으로 결정
  - 투자 시간 → 성실성/신경증에 의해 결정
  - 종목 선택 → 개방성/위험선호도에 의해 결정
  - 투자 비율 → 위험선호도/손실회피에 의해 결정
  - 매매 전략 → 성격 조합에 의해 프로필 결정
- **감정 시스템**: 수익/손실에 따라 스트레스, 자신감, 피로도가 변동하며 투자 행동에 영향

## 인사 시스템

| 항목 | 설명 |
|------|------|
| **채용** | 자산 증가 시 랜덤으로 신규 인원 충원 |
| **해고** | 자산 감소 시 인원 감축 |
| **승진/강등** | 성과에 따라 직급 변동 |
| **성과 평가** | CEO AI가 각 직원의 투자 수익률 기반으로 평가 |

## 프로젝트 진행 단계

1. 자료 조사 (AI 트렌드 + 심리학 모델)
2. 프로젝트 가능 여부 판단
3. 최적 기술 스택 선정
4. 비용 계산
