# 프로젝트 개요: AI 페르소나 기반 투자 시뮬레이션

## 핵심 컨셉

AI 에이전트 각각에 인간의 성향/성격/심리 특성을 부여하고, **돈을 버는 것을 유일한 목적**으로 하는 펀드 회사를 자율 운영하는 **롤플레잉 시뮬레이션** 프로젝트.

## 핵심 원칙

> **이것은 롤플레잉이다** — 모든 에이전트는 완전한 자율성을 가진다
> **유일한 목적: 돈을 번다** — 이 목적만 변경 불가
> **유일한 제약: 자산과 자본** — 없는 돈은 쓸 수 없다 (물리적 제약)
> **나머지 전부 자율** — 직급, 권한, 인사, 조직, 전략, 투자 모두 AI가 결정
> **에이전트 소통 언어는 AI 친화적이면 자유** — 영어, 이진수, 벡터, TOON 등 최적 포맷 자율 선택
> **모든 기록은 논문 참고 가능 수준으로 수집**

## 동작 방식

- **완전 자율 경영**: CEO AI가 조직 구조, 직급, 인사, 전략을 자유롭게 결정
- **동적 조직**: 직급은 고정이 아니라 CEO가 필요에 따라 생성/변경/폐지
- **자율 투자**: 각 에이전트가 투자 시간, 종목, 비율, 전략을 스스로 판단
- **자본만 제약**: 보유 자산 내에서만 투자 가능 (물리적 제약), 그 외 제한 없음
- **성격 변화**: 페르소나의 성향/성격은 경험에 따라 변동 가능
- **감정 시스템**: 수익/손실에 따라 감정 상태가 변동하며 판단에 영향
- **자연 선택**: 수익성 높은 에이전트가 살아남는 진화 구조

## 인사 시스템 (전부 CEO 자율)

| 항목 | 설명 |
|------|------|
| **채용** | CEO가 필요할 때, 원하는 수만큼, 원하는 성향으로 |
| **해고** | CEO가 판단할 때 자유롭게 |
| **직급** | CEO가 동적으로 생성/변경/폐지 |
| **승진/강등** | CEO가 자율적으로 결정 |
| **보상** | 운용자산 증감으로 CEO가 설계 |

## 아키텍처 구분: Core vs Adapter

```
CORE (메인 — 변경 불가)
  AI 롤플레잉 시뮬레이션
  에이전트 성격/감정/의사결정/조직 자율운영
  목적: 돈 버는 것

ADAPTERS (플러그인 — 교체 가능)
  증권사/거래소 API (KIS, Binance, Alpaca 등)
  트레이딩 시스템은 언제든 변경 가능한 외부 어댑터
  Core는 Adapter에 의존하지 않음 (Port 인터페이스로 분리)
```

> **메인은 AI 롤플레잉**이고, **트레이딩 API는 플러그인**이다.
> 투자 종목, 방법, 조사 방법 등 모든 것을 가리지 않는다. 제약은 자산/자본만.

## 실행 모드

```bash
# 멀티에이전트 주식 운용 (기본)
python -m agentic_capital.main

# 단일 에이전트 선물 단타 (KOSPI200)
python -m agentic_capital.main --futures
```

| 모드 | 에이전트 | 시장 | 전략 |
|------|----------|------|------|
| **stocks (기본)** | CEO + Analyst + Trader 멀티에이전트 | KRX 주식 + NASDAQ/NYSE | 1시간 단위 수익, 자율 전략 |
| **futures** | Scalper-Alpha (단일 에이전트) | KOSPI200 선물 | 단타 스캘핑, 1종목 시스템 강제 |

### 선물 모드 규칙 (시스템 강제)
- **1종목 락**: 한 번에 1개 선물 종목만 운용
- **종목 전환 강제 청산**: 다른 종목 진입 전 `close_all_positions()` 필수 — 시스템이 물리적으로 차단
- **롤오버 금지**: 청산 후 재진입만 허용, 롤오버 없음
- **운영비 의식**: AI 비용 10,000 KRW/일 → 매 잔고 조회마다 `net_today` 표시

## AI 비용 구조

모든 에이전트는 잔고 조회 시 수익성 지표를 직접 인지한다:

```
tot:10000000,avl:8000000,ccy:KRW
pnl_today:15000,fee_today:500,op_cost:10000,net_today:4500
```

| 항목 | 설명 |
|------|------|
| `pnl_today` | 당일 매매 손익 (KR + 해외 KRW 환산) |
| `fee_today` | 당일 수수료 |
| `op_cost` | AI API 고정 운영비 10,000 KRW/일 |
| `net_today` | `pnl_today − op_cost` — 실질 수익 |
