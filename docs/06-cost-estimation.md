# 비용 계산

## 가정 조건

| 항목 | 값 |
|------|-----|
| 회사 구성원 | 에이전트 총 10명 (CEO 1명, 사원 9명) |
| 일일 활동 | 각 에이전트 하루 10번 판단 (API 호출) |
| 1회 호출 토큰 | 입력 2,000 토큰 + 출력 500 토큰 |
| 일일 총 토큰 | 약 250,000 토큰 |
| 모델 비율 | Pro 10% / Flash 90% |

## 월별/일별 예상 운영 비용 (USD)

| 항목 | 일일 | 월간 (30일) | 비고 |
|------|------|-------------|------|
| AI API (Pro급) | ~$0.50 | ~$15.00 | CEO 복잡한 평가 및 최종 결정 |
| AI API (Flash급) | ~$0.10 | ~$3.00 | 일반 사원 반복적 투자 판단 |
| 클라우드 서버 | ~$1.00 | ~$30.00 | AWS EC2 t3.medium 급 |
| 데이터베이스 | ~$0.80 | ~$25.00 | AWS RDS PostgreSQL + 스토리지 |
| **합계** | **~$2.40** | **~$73.00** | 초기 개발 및 소규모 테스트 기준 |

## 비용 절감 포인트

- Flash 모델 비율을 높일수록 API 비용 절감
- 초기에는 모의투자(Paper Trading)로 불필요한 실거래 비용 방지
- 에이전트 수를 단계적으로 늘려가며 비용 추이 관찰
